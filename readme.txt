This is the readme for the models associated with the paper:

Muddapu, V.R., Vijaykumar, K., Ramakrishnan, K., Chakravarthy, V.S., 2020. 
A Computational Model of Levodopa-Induced Toxicity in Substantia Nigra Pars Compacta 
in Parkinson’s Disease. bioRxiv. 
https://doi.org/10.1101/2020.04.05.026807

This archive was contributed by Vignayanandam R. Muddapu.

How to run the model

run RUN_LIT_model.m

See into RUN_LIT_model.m file to reproduce the different results from the paper
